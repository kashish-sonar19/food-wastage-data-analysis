#  Community Food Share Portal

A centralized, user-friendly portal designed to bridge the gap between food donors and communities in need. This system actively manages food inventory, tracks distribution, and empowers administrators to maintain database integrity.

## Core Functionalities
- **Dashboard Overview:** High-level metrics showcasing available items, registered NGOs, and active donors.
- **Distribution Insights:** Modern Sunburst and horizontal bar charts mapping the hierarchy of food availability.
- **Food Directory:** Seamless horizontal filtering system to locate specific dietary needs across different cities.
- **System Admin Control:** An expander-based secure admin panel for full CRUD (Insert, Modify, Drop) operations on the database.

##  Technologies Used
- **Language:** Python
- **Web Framework:** Streamlit
- **Database:** SQLite
- **Libraries:** Pandas, Plotly, SQLAlchemy

##  Mission
To minimize urban food waste by providing a transparent, real-time inventory system that connects those who have excess with those who need it the most.
